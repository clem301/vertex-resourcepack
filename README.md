# Vertex Resource Pack

Resource pack officiel du serveur Minecraft Vertex.

## Installation

1. Télécharger [PetrolierResourcePack.zip](https://github.com/VOTRE-USERNAME/vertex-resourcepack/raw/main/PetrolierResourcePack.zip)
2. Options → Resource Packs → Open Pack Folder
3. Copier le ZIP dans ce dossier
4. Activer le pack dans Minecraft

## Contenu

- Modèle 3D personnalisé pour le pétrolier
- Textures custom

## Serveur

Le pack se télécharge automatiquement en rejoignant le serveur.
